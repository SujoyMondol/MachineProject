# Jack in the Box Animation Simulator

## Overview

This project is a C++ application built using **wxWidgets** that simulates a Jack in the Box through a frame-based animation system. The project demonstrates object-oriented design principles, hierarchical scene composition, and event-driven programming while rendering an interactive animated machine.

## Features

* Built in **C++**
* GUI developed with **wxWidgets**
* Frame-based animation using keyframes and timelines
* Hierarchical object composition for animated machine components
* Object-oriented design using inheritance and polymorphism
* Event-driven user interaction

## Technologies

* C++
* wxWidgets
* Object-Oriented Programming (OOP)

## Project Structure

The application consists of multiple classes representing machine components that work together to produce the animation. A timeline controls animation playback while individual objects update their positions and transformations based on the current animation frame.
